NumPy Introduction

NumPy is a Python library for working with arrays, and provides high-performance multidimensional array objects and tools for working with these arrays.

What is NumPy?

NumPy stands for Numerical Python. It is an open-source Python library that provides support for large, multi-dimensional arrays and matrices, along with a large collection of high-level mathematical functions to operate on these arrays.

ℹ

NumPy arrays are stored in contiguous blocks of memory, making operations much faster than Python lists which store pointers to objects scattered in memory.

Why NumPy?

In Python we have lists that serve the purpose of arrays, but they are slow to process. NumPy aims to provide an array object that is up to 50x faster than traditional Python lists.

# Python list vs NumPy — performance demo
import numpy as np
import time

py_list = list(range(1_000_000))
np_arr  = np.arange(1_000_000)

# Python list
t = time.time()
result = [x ** 2 for x in py_list]
print(f"List: {time.time()-t:.4f}s")

# NumPy
t = time.time()
result = np_arr ** 2
print(f"NumPy: {time.time()-t:.4f}s")
Output
List: 0.1823s
NumPy: 0.0041s

print(mask)       # [False False False  True  True  True]

print(arr[arr % 2 == 0])  # [2 4 6]

Copy vs View

Understanding the difference between a copy and a view is essential to avoid bugs when modifying arrays.

View — Shared Memory

A view does not own the data. Changes to the view affect the original array.

orig = np.array([1, 2, 3, 4, 5])
view = orig.view()

view[0] = 99
print(orig)  # [99  2  3  4  5]  ← original changed!
print(view.base is orig)  # True

Copy — Independent Data

orig = np.array([1, 2, 3, 4, 5])
copy = orig.copy()

copy[0] = 99
print(orig)  # [1 2 3 4 5]  ← original unchanged
print(copy.base is None)  # True
⚠

Slices return views — not copies. If you want to modify a slice without affecting the original, always call .copy() explicitly.

Array Reshape

Change the shape of an array without changing the data using reshape(), flatten(), and ravel().

reshape()

arr = np.arange(1, 13)         # [1..12]

a23  = arr.reshape(2, 6)       # 2 rows, 6 cols
a234 = arr.reshape(2, 3, 2)    # 3-D
auto = arr.reshape(3, -1)       # -1 infers the missing dim

print(a23)
print(auto.shape)   # (3, 4)

Flattening

arr = np.array([[1,2],[3,4],[5,6]])

print(arr.flatten())  # [1 2 3 4 5 6] — copy
print(arr.ravel())    # [1 2 3 4 5 6] — view when possible
✓

Use reshape(-1) or ravel() for a view, and flatten() when you need an independent copy.

ufunc Introduction

Universal functions (ufuncs) operate element-wise on ndarrays, support broadcasting, type casting, and other special features.

What are ufuncs?

ufuncs are NumPy functions that operate element-by-element on arrays. They are implemented in C, making them very fast. NumPy provides over 60 built-in ufuncs.

⚡
Vectorized
Replace slow Python loops with fast C operations
📡
Broadcasting
Operate on arrays of different (compatible) shapes
🔄
Type Casting
Automatic conversion of dtypes where needed
# ufunc example: np.add is a ufunc
a = np.array([1,2,3])
b = np.array([4,5,6])

print(np.add(a, b))         # [5 7 9]
print(a + b)               # same — uses np.add internally
print(type(np.add))         # <class 'numpy.ufunc'>
Home/Create ufunc
Universal Functions

Create ufunc

Create custom ufuncs using np.frompyfunc() to wrap Python functions for element-wise array operations.

# Define a Python function
def my_add(x, y):
    return x + y

# Convert to ufunc (nin=2 inputs, nout=1 output)
my_add_ufunc = np.frompyfunc(my_add, 2, 1)

a = np.array([1,2,3])
b = np.array([4,5,6])
print(my_add_ufunc(a, b))   # [5 7 9]

# Check type
print(type(my_add_ufunc))    # <class 'numpy.ufunc'>

Trigonometric Functions

NumPy provides sin, cos, tan, and their inverses, all operating on radians.

FunctionDescription
np.sin(x)Sine (radians)
np.cos(x)Cosine (radians)
np.tan(x)Tangent (radians)
np.arcsin(x)Inverse sine → radians
np.arccos(x)Inverse cosine → radians
np.arctan(x)Inverse tangent → radians
np.deg2rad(x)Degrees to radians
np.rad2deg(x)Radians to degrees
angles = np.array([0, 30, 45, 60, 90])
rad = np.deg2rad(angles)

print(np.sin(rad))
# [0.    0.5   0.707 0.866 1.   ]

# hypotenuse using hypot
print(np.hypot(3, 4))   # 5.0

Hyperbolic Functions

Hyperbolic analogues of trigonometric functions used in physics, engineering, and deep learning activations.

FunctionDescription
np.sinh(x)Hyperbolic sine
np.cosh(x)Hyperbolic cosine
np.tanh(x)Hyperbolic tangent — activation function
np.arcsinh(x)Inverse hyperbolic sine
np.arccosh(x)Inverse hyperbolic cosine
np.arctanh(x)Inverse hyperbolic tangent
x = np.array([0, 0.5, 1.0, 2.0])
print(np.sinh(x))   # [0.    0.521 1.175 3.627]
print(np.tanh(x))   # [0.    0.462 0.762 0.964]

Set Operations

Perform set-theoretic operations on 1-D arrays: union, intersection, difference, and unique.

FunctionDescription
np.unique(a)Unique sorted elements
np.union1d(a, b)Union (all unique elements)
np.intersect1d(a, b)Common elements
np.setdiff1d(a, b)In a but not in b
np.setxor1d(a, b)Exclusive-or (not in both)
np.in1d(a, b)Test membership elementwise
a = np.array([1,2,3,4,3,2])
b = np.array([3,4,5,6])

print(np.unique(a))              # [1 2 3 4]
print(np.union1d(a,b))          # [1 2 3 4 5 6]
print(np.intersect1d(a,b))     # [3 4]
print(np.setdiff1d(a,b))       # [1 2]
print(np.setxor1d(a,b))        # [1 2 5 6]
